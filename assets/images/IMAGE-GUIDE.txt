Clean Sweep Image Guide
=======================

The official Clean Sweep logo is stored at:

assets/logo/full-logo-transparent.png

Use the transparent full logo in footers and the transparent symbol/wordmark crops in compact headers and mobile navigation.
Keep logo files optimized for the web and avoid spaces in file names.

The layout is designed to preserve the logo's proportions without stretching.

Photography
-----------

AI-generated placeholder images have been removed from the project.
Only add original client-provided photography here.

Recommended dimensions:

- Hero or wide background photo: 1920 x 1200
- Section or service photo: 1400 x 900
- Detail photo: 1400 x 1000

Use sharp, professional images of the client's actual sweeping services, equipment, clean commercial properties, parking lots, or maintained exterior areas.
Avoid fake company logos, watermarks, readable license plates, or identifiable people unless the client has approved their use.

How to replace images
---------------------

1. Export each approved client image with a clean lowercase web file name.
2. Place the file in this folder: assets/images/
3. Keep images optimized for the web. WebP is recommended for photos.
4. Update the relevant HTML or CSS reference.
5. Refresh the browser and confirm the image is not stretched or cropped poorly.
